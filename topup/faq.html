FAQ

Bagaimana Tata Cara Melakukan Top Up?

Ikuti intruksi dari nomor 1 hingga akhir.

Bagaimana Jika Saya Ingin Menanyakan Sesuatu ke Admin?

Pembeli dapat menanyakan mengenai suatu permasalahan ke Admin melalui customer service(support).

Kenapa Order Saya Belum Diproses?

Order akan dilayani oleh Admin satu per satu, jika terdapat kendala maka Admin akan menghubungi cosumer.

Pada Pukul Berapa Order dapat Diproses?

Pada pukul 9.00 - 21.00

Berapa Lama Order akan Diproses?

Jika tidak ada kendala, maka kurang lebih 1 - 10 menit.
